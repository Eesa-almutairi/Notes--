Some of Important principles for Intro to Programming Nanodegree

What is a Nanodegree

Nanodegree is a path and this path starts with nothing! (i.e you start your journey building an HTML File
with blanck page).

As a progress of this journey, you are going to gain three important things:
1- New skills.
2- New knowledge.
3- New ways of thinking.

How to think like a programmer?

    there are several ways of thinking:
  • Procedural thinking:
    • this tpye of thinking means to creat clear instructions for computer to follow.
  • Abstract thinking :
    • it involves finding similarities among different things.The purpose of this one is
    to avoid repetition of work.
  • Systemmatic thinking:
    • It means to break big problems into small pieces to make typing code pretty easy.
  • Theoritical empathy:
    • It involves understanding how the computer thinks, (i.e. imagination of how the code you are tyoing
    will look like after running the program)
  • Debugging:
    • Is a Systemmatic process to identify the causes of failure for HTML file that doesn't
    work.
    (bug means mistake, debugging is the process of finding mistakes).

Here is a Systemmatic process to fix a problem:

1- collect evidence. ( asking yourself: What is made this program fail?).
2- Generate theories. (asking yourself: what may have caused this problem?).
3- Test those theories.( asking yourself: if my theories are correct, how could I find out?).
4- Fix the problem.

The World Wide Web (WWW):

What is the World Wide Web?
it is simply collection of HTML documents.
While HTML stands for Hyper Texts Markup Language.

How user,computer,browser,internet,HTTP and server work together?

First, the user use the computer to open a web page called ( Google for example) using
the browsers (e.g. firefox,internt explorer,chrome etc ). second the browser will communicate with google
server to open the HTML document that belongs to Google using HTTP. Finally, the server
will respond to the user request by displaying the document which looks like this.

Definitions of World Wide Web terms

User: is a person who is using the computer.He is the main part of WWW.

browser: is an application that communicat and requsts HTML documents
from the server.
Internet: is like a bridge between the browser and server to transfer documents.It is consider
as the world's larges computer network.
Server: is a computer, but instead of hosting and displaying documents, it is only hostign files and
documents.

Block Vs Inline

Inline: An inline element does not start on a new line and only takes up
as much width as necessary. Examples: {a}anchor,{span} ,{br}break ,{img}image
Block: A block-level element always starts on a new line and takes up the full width available
.Examples: {div} ,{P}paragraph,{address},{main}
